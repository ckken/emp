# EMP Share

EMP Share is a module sharing solution based on Module Federation. It allows you to share components, functions, and other resources between different projects.

## Features

- **Type Safety**: Provides TypeScript type definitions for shared modules
- **Framework Agnostic**: Supports sharing between different frameworks (React, Vue)
- **Easy Integration**: Simple configuration and API for module sharing
- **Performance Optimized**: Efficient loading and caching of shared modules

## Basic Usage

### Host Project Configuration

```typescript
// emp.config.ts
import { defineConfig } from '@empjs/cli'

export default defineConfig({
  server: {
    port: 8001
  },
  moduleFederation: {
    name: 'host',
    filename: 'emp.js',
    exposes: {
      './Button': './src/components/Button'
    },
    shared: ['react', 'react-dom']
  }
})
```

### Remote Project Configuration

```typescript
// emp.config.ts
import { defineConfig } from '@empjs/cli'

export default defineConfig({
  server: {
    port: 8002
  },
  moduleFederation: {
    name: 'remote',
    filename: 'emp.js',
    remotes: {
      host: 'host@http://localhost:8001/emp.js'
    },
    shared: ['react', 'react-dom']
  }
})
```

### Using Shared Modules

```typescript
import { Button } from 'host/Button'

function App() {
  return (
    <div>
      <Button>Shared Button</Button>
    </div>
  )
}
```

## Advanced Features

### Type Generation

EMP automatically generates TypeScript type definitions for shared modules. You can configure type generation in your `emp.config.ts`:

```typescript
export default defineConfig({
  moduleFederation: {
    // ... other config
    typesPath: './src/types'
  }
})
```

### Dynamic Loading

You can dynamically load shared modules using the `import()` syntax:

```typescript
const Button = await import('host/Button')
```

### Versioning

Manage shared module versions using the `shared` configuration:

```typescript
export default defineConfig({
  moduleFederation: {
    shared: {
      react: { requiredVersion: '^18.0.0' },
      'react-dom': { requiredVersion: '^18.0.0' }
    }
  }
})
```

For more detailed information about EMP Share configuration and usage, please refer to the [Configuration Guide](../config/empShare/index.mdx).
