# Configuration

`EMP` provides configuration options similar to webpack. Through this chapter, you can learn how to use `EMP` configuration.

## Configuration File

When you run the `EMP` command-line tool, `EMP` will automatically read the `emp-config.js` file in the current directory.

A basic `EMP` configuration file looks like this:

```ts title="emp-config.js"
import {defineConfig} from '@empjs/cli'
export default defineConfig(store => {
  return {
    chain(chainConfig) {},
  }
})
```
