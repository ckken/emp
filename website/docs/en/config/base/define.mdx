+ Global environment replacement
+ Type: `Record<string, any>`

## defineFix
+ Type: `all` | `esm` | `cjs` | `none`
+ Default: `cjs`

:::tip 
Whether to create process.env for cjs or import.meta.env for esm
all: create both
esm: create import.meta.env
cjs: create process.env
none: create neither
:::

```ts title="emp-config.js"
import {defineConfig} from '@empjs/cli'
export default defineConfig(store => {
  return {
        define:{
            cdnHost:'cdn.com'
        },
        defineFix:'cjs'
    }
  }
})
```
