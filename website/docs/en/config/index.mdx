# Configuration Overview
`EMP` v3 provides configuration options similar to v2. Through this chapter, you can learn how to use `EMP` configurations.

import Overview from '@components/Overview';

<Overview list={OVERVIEW_CONFIGS} />

When you run the `EMP` command line tool, `EMP` will automatically read the `emp-config.js` file in the current path.
A basic `EMP` configuration file is shown below:

```ts title="emp-config.js"
import {defineConfig} from '@empjs/cli'
export default defineConfig(store => {
  return {}
})
```
